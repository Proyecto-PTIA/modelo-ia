# S.I.R.I. API - Inteligencia Artificial para Recetas
Esta es la API de backend para el sistema de recomendación de recetas basado en imágenes.

## 🚀 Endpoints
* **GET /**: Verificación de estado.
* **POST /detect-dish**: Envía una imagen en el campo `foto` para obtener el plato e ingredientes.
